# API概览

本产品(`通义数据挖掘`)使用DashScope HTTP接口对外提供服务。调用接口需要先申请API KEY。

## **API目录**

**API名称**

**API概述**

[文档上传](https://help.aliyun.com/zh/model-studio/document-upload)

上传并提交文档

[信息抽取](https://help.aliyun.com/zh/model-studio/document-information-extraction)

根据指定字段列表，在上传文档中进行搜索并抽取

[文档内容审核](https://help.aliyun.com/zh/model-studio/document-content-audit)

根据指定审核规则，快速捕捉内容不合规点，高精度输出审核结果

[打标分类](https://help.aliyun.com/zh/model-studio/document-tagging)

根据指定分类标签，将上传文档进行分类打标

[摘要生成](https://help.aliyun.com/zh/model-studio/document-summary-generation)

对文档进行一段话总结

[文档删除](https://help.aliyun.com/zh/model-studio/document-delete)

删除已上传文档
