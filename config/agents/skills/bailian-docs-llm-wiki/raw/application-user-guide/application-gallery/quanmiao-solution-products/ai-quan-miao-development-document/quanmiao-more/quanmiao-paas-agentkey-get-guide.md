# 全妙PaaS AgentKey 获取指南

本文主要介绍调用全妙接口中AgentKey如何获取。

## **AgentKey**

AgentKey 是全妙历史接口中的关键标识字段，用于指定阿里云百炼应用广场中“全妙”应用所对应的业务空间，从而实现对该空间下资源的精准访问。

## **获取步骤**

1.  根据需求开通全妙的产品；
    
2.  进入全妙SaaS页面；
    
    1.  妙笔/妙策SaaS地址：[https://aimiaobi.console.aliyun.com/?product\_code=g\_broadscope\_media&from=bailian#/audit](https://aimiaobi.console.aliyun.com/?product_code=g_broadscope_media&from=bailian#/audit)；
        
    2.  妙搜SaaS地址：[https://aimiaobi.console.aliyun.com/?product\_code=g\_broadscope\_search#/search\_copilot\_reference](https://aimiaobi.console.aliyun.com/?product_code=g_broadscope_search#/search_copilot_reference)；
        
3.  鼠标悬浮至账号上，即可获取AgentKey。
