# release notes

百炼平台 release notes 汇总了模型平台功能更新与模型上下架动态，涵盖 2024 年至今的功能迭代、计费调整、新模型上架与下线机制。开发者可在此跟踪 API 能力扩展（如 Responses API 异步调用、模型导入 API）、新模型规格（千问3.6/3.7、DeepSeek-V4、Kimi K2.7 等）以及关键的计费变化。详细功能动态与模型清单参见 [模型平台功能更新](../../raw/model-user-guide/release-notes/model-release-notes.md) 与 [模型上下架与更新](../../raw/model-user-guide/release-notes/newly-released-models.md)。

## 功能动态概览

百炼平台按月发布功能更新，主要围绕以下模块展开，最新进展（截至 2026 年 6 月）要点如下。

### 模型调优与训练

- **强化学习训练（邀约制）**：2026 年 5 月 31 日上线，支持基于奖励信号优化模型策略。
- **多模态调优扩展**：新增图像生成模型（Wan/Wanx）、视觉理解（VL）模型、视频生成模型类型的 SFT 训练支持。
- **安全合规强化**：2026 年 5 月 4 日上线 0 代码大模型安全合规强化训练流程。
- **模型压缩**：2026 年 5 月 25 日上线，通过量化算法将全精度微调模型转为低精度版本以降低部署成本。

### API 与应用开发

- **Responses API 异步调用**：2026 年 6 月 1 日上线，通过 `background=true` 提交长耗时任务并轮询结果。
- **模型导入 API**：2026 年 6 月 3 日上线，涵盖导入任务与自定义模型对象的完整 API；国际站同时支持从 OSS 导入 LoRA 微调模型。
- **临时 API Key**：2026 年 6 月 3 日新增生成临时 API Key 文档，用于不可信环境避免永久密钥泄露。
- **新版智能体应用 DashScope API**：2026 年 5 月 11 日首发，支持单轮/多轮、流式、文件问答与视觉理解。
- **Prompt 工程 API**：2026 年 4 月 7 日上线，提供 Prompt 模板管理能力。

### 应用与生态

- **UI 设计器**：2026 年 4 月 1 日上线，集成魔笔低代码能力，可视化拖放构建网页 UI 应用。
- **记忆库 Memory 2.0**：2026 年 3 月 20 日上线，自动从对话提取记忆片段与用户画像，支持多应用共享同一记忆库。
- **官方 MCP 服务**：2026 年 2 月 6 日上线，支持平台内集成与外部第三方接入（如 Amap Maps）。
- **多模态交互开发套件**：陆续推出服务端 Java SDK、移动端 Android SDK、Linux C++ SDK、RTOS C SDK（License 模式）等多端 SDK。

### 模型观测与评测

- **模型评测排行榜**：2026 年 6 月 9 日新增 Leaderboard 管理入口与综合评测能力，支持 BLEU_4 等评分方法；可一次选择多个模型基于多维度生成对比榜单。
- **知识库日志与监控**：2026 年 4 月 24 日上线，检索调用全量投递至 SLS 日志服务，支持审计/排查/统计/告警。
- **异步任务事件回调**：2026 年 4 月 23 日支持通过事件总线 EventBridge 主动推送任务完成事件，无需轮询。

更多按月排列的功能点请参见 [模型平台功能更新](../../raw/model-user-guide/release-notes/model-release-notes.md)。

## 计费与定价调整

平台多次调整模型计费策略，开发者需关注以下关键变化：

- **qwen-max 大幅降价（2025 年 2 月）**：输入价格降低 88%，输出价格降低 84%。实时调用输入从 0.02 元/千 Token 降至 0.0024 元/千 Token。
- **千问 VL 系列降价（2024 年 12 月）**：视觉理解模型降价最高可达 85%。
- **DeepSeek 系列转计费（2025 年 2 月）**：部分模型由免费体验转为计费。
- **deepseek-v4-pro 缓存计费调整（2026 年 4 月）**：`cached_token` 单价调整为 1 元/百万 token，标准 `input_token` 单价不变。
- **按模型单元（MU）部署计费**：2025 年 10 月 24 日新增按时间计费的部署方式，核心优势为可灵活调整性能、高稳定性与可预测的固定成本。
- **免费额度用完即停**：2025 年 7 月 29 日上线，启用后新人免费额度耗尽返回 `AllocationQuota.FreeTierOnly` 错误，避免产生额外费用。

最新计费详情以[百炼控制台](https://bailian.console.aliyun.com/cn-beijing?tab=model#/model-market/all)为准。

## 关键能力演进（2024–2025）

早期能力演进为当前生态奠定基础，主要包括：

- **Context Cache（2024 年 12 月）**：减少推理时重复运算量，提升响应速度并降低成本，首发支持 qwen-plus。
- **模型观测（2025 年 1 月）**：监测模型使用情况与性能变化，后续陆续增加高级监控模式（分钟级低延时、4xx/5xx 分别统计）与告警通知。
- **Batch 任务通知（2024 年 12 月）**：支持 Callback 回调与 EventBridge 消息两种完成通知方式。
- **DPO 偏好训练（2025 年 9 月）**：千问3-32B/14B/8B 与千问2.5-72B/32B/14B/8B 新增 DPO 训练，通过负反馈降低幻觉。
- **免费额度与用量看板（2025 年 12 月）**：集中展示各模型免费额度（已用量与剩余额度）及调用量统计。

## 使用建议

1. **接入新模型前先查上下架清单**：模型规格与能力（思考模式、模态支持、上下文长度）更新频繁，预览版与正式版行为可能不同。
2. **长耗时任务用异步调用**：Responses API 异步模式或 EventBridge 事件回调可避免轮询。
3. **关注计费变化**：尤其是免费转计费节点，建议开启"免费额度用完即停"避免意外扣费。
4. **不可信环境使用临时 API Key**：避免永久 API Key 泄露。
5. **生产部署优先考虑 MU 模式**：固定成本与高稳定性更适合稳定负载场景。
