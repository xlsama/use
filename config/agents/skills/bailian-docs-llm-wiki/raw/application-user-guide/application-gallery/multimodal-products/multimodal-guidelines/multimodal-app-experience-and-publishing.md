# 应用体验与发布

完成应用配置后，您可以在页面实时体验和调试效果。确认运行效果后，发布应用，即可在线上服务中成功调用。

## **应用体验**

完成配置后，点击下方**立即运行**即可体验。

**说明**

请佩戴耳机，并在安静环境下测试，对话体验更佳。

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8855897471/p958689.png)

-   如需配置API Key，请按照以下步骤操作。
    
    -   若未授权，需先进行授权操作；授权成功后即可创建API Key。![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8855897471/p958690.png)
        
    -   若已授权，则可直接创建API Key。
        
    
    ![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8855897471/p958692.png)
    
    ![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8855897471/p958686.png)
    

-   当您开启语音交互时，可体验实时语音对话；关闭语音交互后，可体验文本对话。如果在多模态交互应用中选择实时视频对话Agent，还可以体验实时视频对话。
    
-   在首次体验前，需要您为当前页面授予摄像头、麦克风的使用权限。![image.jpeg](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8855897471/p958687.jpeg)
    
-   若配置应用时未开启语音交互，右侧仅支持文本交互体验。![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3986570571/p977082.png)
    
-   管控台页面支持显示端到端延迟信息。端到端首包延迟，可以理解为「ASR识别出最后一个字 - TTS播报第一个字」之间的时间间隔，也就是用户体感的等待时间。
    
    -   ![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3759579671/p1041272.png)
        

## **应用发布**

确认运行效果后，点击**发布**，在**发布版本**对话框中完成**版本描述**信息填写后，单击**确定**，当前配置将生效。您可以使用当前应用对应的唯一ID开发相应能力。

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/8855897471/p958688.png)

发布后，您可以继续修改配置。未发布的版本将不会影响线上服务。
