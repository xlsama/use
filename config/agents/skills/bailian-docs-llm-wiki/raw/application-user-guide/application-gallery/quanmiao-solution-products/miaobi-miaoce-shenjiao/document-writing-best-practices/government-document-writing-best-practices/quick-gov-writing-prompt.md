# 快速写一篇政务稿（prompt一步式撰写）

文章主题（prompt）

请以政府的口吻，帮我写一篇复函，篇幅为中篇，标题为：关于《关于搬迁老城区儿童园、动物园的建议》的回复函

文章标题

关于《关于搬迁老城区儿童园、动物园的建议》的回复函

## 具体操作

-   首先选择“创作政务公文”
    

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3881943071/p750214.png)

-   随后选择“直接生成”
    

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4881943071/p750213.png)

-   按要求填写【主题】和【标题】内容，并且可以根据实际需求选择【文体】、【文章篇幅】、【输出语言】、【生成篇数】，完成后点击“开始写作”。
    

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3881943071/p750215.png)

-   等待系统创作内容
    

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3881943071/p750216.png)

-   查看系统创作的内容是否符合预期，如果符合预期，点击“采纳”。
    

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3881943071/p750220.png)

-   生成结果未达到预期的修改
    

1).如果对文章的引用来源不满意，可点击【结果干预】对引用源进行删改和【重新搜索】内容引用源功能，删改后可点击【重新生成】新文章；

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3881943071/p750218.png)

2).如对整体结果不满意，可使用【换一换】功能对文章内容进行【直接换】（重写）、【扩写】或【缩写】等操作

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4881943071/p750219.png)

3).当您已经采纳并排版后，发现还是不满意，可以选择您需要修改的文字，在弹出的浮窗中可以进行快捷生成，具体包括：生成小标题、摘要、续写、扩写、缩写

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4881943071/p750221.png)

**说明**

续写：根据选中原文继续延伸 扩写：根据选中原文改动词汇，扩大字数 缩写：根据选中原文改动词汇，缩减字数

## “搜索素材”丰富文章内容

-   点击“搜索素材”，进入到素材搜索界面![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4881943071/p750228.png)
    
-   随后在搜索框内输入您要搜索的内容即可，如：关于搬迁老城区儿童园、动物园的相关文章
    

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4881943071/p750230.png)

-   您可以将符合您预期的素材内容加入素材库
    

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/3881943071/p750229.png)
