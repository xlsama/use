# 计费说明（视频混剪）

本文档主要介绍全妙-视频混剪的计费方式。

## **计费说明**

本文档为视频混剪服务的后付费计费说明。计费按上传的视频源素材时长计算，系统默认提供 2 个并发任务额度，即最多同时运行 2 个混剪任务，如需增加并行任务数量，可以通过扩并发计费项实现。

## **开通产品**

开通产品：点击[视频混剪开通页面](https://common-buy.aliyun.com/?commodityCode=sfm_quanmiaoAPI_public_cn)即可开通视频混剪服务。

扩并发说明

如需提升任务并行处理能力，可通过购买“扩并发”计费项增加并发额度。扩并发费用根据实际上传的视频素材时长计费。
