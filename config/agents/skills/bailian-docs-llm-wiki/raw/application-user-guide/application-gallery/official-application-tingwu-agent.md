# 官方应用-通义听悟Agent

通义听悟 Agent 深度融合智能语音AI与千问大模型技术，面向企业办公（如：会议、面试、培训）、在线教育、销售服务洞察、汽车行业产-供-销（如：购车客户画像、工业语音质量等）分析等垂直行业场景，提供针对性音视频智能分析解决方案。通过“低门槛调用+高精度处理”双引擎驱动，助力企业快速挖掘业务数据价值。帮助企业客户构建跨行业智能化生态，帮助客户降本增效并升级商业模式。
