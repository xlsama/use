# 2024年2月28更新-AI全妙系列 V2.2

## 新增功能如下​

1.  热点管理升级为【**AI妙策**】，整个产品能力更加丰满
    
    1.  以热点话题榜为例，可以利用大模型链路做话题分析
        
    2.  提供 “**网友、热门、时效、新颖度** ”四个维度的选题切入点分析
        
    3.  围绕给出的切入点，可生成策划提案（包括文本格式和Xmind格式），并可联动【**AI妙笔**】直接进行后续的文案创作
        
2.  【**AI妙搜**】做了大幅升级：
    
    1.  从之前基于文本的增强搜索，升级为面向文、图、音视频的多模态增强搜索
        
    2.  引入multi-agent架构，支持**copilot形态下的高级搜索形式**，更好地处理复杂的搜索任务
        
3.  把【**AI妙笔**】中和营销文案相关的写作能力单独拎出
    
4.  【**AI妙笔**】模型及文体迭代
    
    1.  政务公文：更新了行业模型，修复重复性、内容控制问题；并在 “直接生成”中增加了“**工作总结**”文体
        
    2.  传媒文稿：更新了传媒模型，基于此增加了“新闻简讯”、“演讲稿”、“解说词”三个新文体
        
5.  【**AI妙笔**】在公共云上的版本支持自定义存储功能（支持OSS对象存储）
