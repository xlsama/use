# Prompt自动优化

当手动编写高质量的 Prompt 耗时且需要大量专业知识时，可使用阿里云百炼的 Prompt 自动优化功能。该功能通过大模型对输入的原始 Prompt 进行分析和重写，旨在生成结构更优、指令更清晰、效果更稳定的新版本，从而提升大模型应用的性能和可靠性。

## **工作原理**

Prompt自动优化功能本质上是利用一个大模型来重构用户提交的 Prompt。优化过程包含但不限于以下几种策略：

-   **结构重组：**调整 Prompt 的整体结构，使其更符合逻辑。
    
-   **角色扮演引导：**为模型设定一个明确的专家角色（如“你是一名资深软件工程师”）。
    
-   **指令增强：**将模糊的指令具体化、步骤化。
    
-   **安全与边界注入：**增加关于输出格式、内容限制等边界条件的说明。
    

## 优化 Prompt

以下步骤介绍如何在阿里云百炼控制台自动优化单个 Prompt。

1.  **进入优化功能：访问****应用开发** > **组件管理** > **提示词**页面，在右上角单击并进入**[自动优化](https://bailian.console.aliyun.com/?tab=app#/component-manage/prompt/optimize)**页面。
    
2.  **输入原始 Prompt：**在**原始prompt**输入框中，粘贴或输入需要优化的Prompt。
    
3.  **执行优化并查看结果：**单击**优化**，系统将调用大模型进行处理，并在**优化后prompt**中展示生成的新版本。
    
4.  **使用优化后Prompt：**根据需要，可对优化结果执行以下操作：
    
    -   **复制**：单击**复制**，将优化后的Prompt内容用于其他场景。
        
    -   **保存为模板**：单击**保存为模板**，将优化后Prompt存入提示词库，便于后续管理、版本控制和团队共享。关于模板管理的更多信息，请参见[Prompt模板概述](https://help.aliyun.com/zh/model-studio/prompt-template#531029ffb065j)。

## 常见问题

1.  **我的 Prompt 数据是否安全？会被用于模型训练吗？**
    
    不会。阿里云严格遵守数据隐私政策，您提交用于优化的数据不会被存储或用于任何模型的训练。
    
2.  **Prompt 自动优化功能如何计费？**
    
    Prompt 自动优化功能不计费。
    
3.  **为什么我的 Prompt 优化失败了？**
    
    失败可能由多种原因导致：
    
    -   输入内容过长，超出了模型的 Token 限制。
        
    -   输入内容触发了模型的安全或内容审核策略。
        
    -   网络问题或服务临时不可用。请稍后重试。
