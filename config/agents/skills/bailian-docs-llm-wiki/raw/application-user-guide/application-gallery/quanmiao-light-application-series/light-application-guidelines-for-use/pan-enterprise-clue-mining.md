# 泛企业线索挖掘

泛企业线索挖掘应用支持从社交媒体数据、通话记录等渠道挖掘用户画像及二次销售机会，同时收集竞品和行业趋势相关线索。

## **功能入口**

登录**阿里云百炼大模型服务平台**，在**应用广场**页面，点击[**泛企业线索挖掘**](https://bailian.console.aliyun.com/?spm=a2c4g.11186623.0.0.134172147obHXO#/app/app-market/quanmiao/enterprise-clue-mining)即可进入该轻应用控制台。

## **功能介绍**

### **应用详情**

在**泛企业线索挖掘**应用的**应用详情**页签，您可以查看功能描述、目标客群、最佳实践、计费规则、全妙相关应用推荐五部分内容。

具体包括：功能描述说明从社交媒体数据、通话记录中挖掘用户画像及二次销售机会；目标客群为企业客户的市场、营销、销售团队；最佳实践展示聊天/通话记录素材经多级标签处理生成打标结果的流程；计费模型有**通义千问-Plus-Latest**（输入¥0.0008/千tokens，输出¥0.002/千tokens）、**通义千问-Max-Latest**（输入¥0.0024/千tokens，输出¥0.0096/千tokens）和**Qwen-Long**（输入¥0.0005/千tokens，输出¥0.002/千tokens）；相关应用推荐包括泛企业VOC挖掘、网络内容安全审核和妙策。

### **效果调试**

点击**效果调试**页签，您可以参考以下步骤设置配置项。在应用页面中，单击**效果调试**页签。

#### **第一步：上传素材**

1.  把待挖掘内容粘贴至文本框，可以是一段对话或通话记录，可参考官方示例填写。进入**泛企业线索挖掘**应用页面，在**销售线索**标签下可选择场景示例，单击**使用本示例**按钮预填任务描述内容。
    
2.  选择线索的模型，默认为**千问-Max**。在**选择模型**下拉框中选择要使用的模型，可选项包括通义千问-Max、通义千问-Plus、通义千问-Max-Latest、通义千问-Plus-Latest、通义千问-Long。配置完成后单击**开始挖掘**。
    

#### **第二步：设置内容标签**

1.  输入**任务描述**。在**任务描述**文本框中输入对该标注任务的描述信息，最多500字。
    
2.  自定义线索标签。
    
    -   按照格式**输入标签名称、标签含义和取值**，支持单个/批量输入。可参考官方提供的**示例**填写。点击**示例**，右侧弹出示例面板，可选择预设场景（如房产销售分析场景、求职线索场景、用户画像分析场景等），勾选所需标签后单击**追加标签**将其添加至标签定义列表。
        
    -   点击**批量输入**，可一键粘贴所有的标签名称、标签含义和取值，自动识别。该表单还包含**任务描述**输入框（最多500字）和**补充业务知识**开关。右侧面板支持一键粘贴标签信息并自动识别，每行为一组，标签名称与取值间通过分隔符隔开，可选**制表符Tab**（适用于Excel）、**@@@**或**自定义**分隔符，粘贴后单击**识别结果并在左侧查看**完成解析。
        
    -   点击**添加自定义标签**，可支持自定义标签，精准挖掘信息。
        
        在弹出的**新增标签**对话框中，填写**标签名称**（限20字符）和**标签含义和取值**（限2000字符），然后单击**确定**。
        
    
3.  **可选**：补充业务知识。
    
    您可以在此文本框中输入业务背景、此任务的注意事项或者任务示例等补充内容。
    
    在**补充业务知识**区域，确保右侧开关已开启（紫色状态），然后在下方文本框中输入内容。
    

#### **第三步：设置输出格式**

您可根据右侧**格式示例**填写输出格式，也可以点击输入框内的**智能生成JSON格式**，一键生成输出格式。

格式示例面板提供 **json** 和 **text** 两种格式页签，展示对应格式的输出模板。确认示例后，单击 **使用本示例替换** 即可将示例内容填入输出格式输入框。

#### **第四步：开始挖掘**

点击**开始挖掘**按钮，挖掘结果将在挖掘结束后显示在右侧，您可复制或下载至本地查阅。单击左侧底部的**开始挖掘**按钮后，右侧面板展示结构化JSON挖掘结果。JSON输出包含已配置的各字段及其提取值，例如：`是否近期买房=Y`、`购房付款方式=纯商业贷款`、`家庭人口数量=至少3人`、`房间数量需求=三房`、`买房投资诉求=自住`、`客户买房类型=改善`、`客户买房预算=150万左右`、`用户核心需求=[学区房]`等。右侧面板顶部提供**展示JSON块**下拉菜单、**下载为Excel**和**复制**按钮，底部显示输入token 696、输出token 110等用量信息。

### **查看API示例**

效果调试完成后，点击**API**页签，您可以查看对应生成的API示例。

**说明**

支持单条实时流式API和批处理API两种调用方式：

-   单条实时流式API：请求一次接口（输入一段文本、发起一次调用），就实时处理并流式返回结果。流式响应协议为Server-Sent Events（SSE）。
    
-   批处理API：输入待挖掘文件或多段文本，后台会创建一个任务，用户需通过任务ID轮询任务的状态、直到任务结束后，返回结果。
    

API 页签提供 Java 和 Python 两种语言的调用示例。Java 示例的 package 为 `org.example`，需导入 `fastjson2`、阿里云认证凭据（auth credentials）、SDK gateway 及 `quanmiaolightapp20240801.AsyncClient` 等依赖。
