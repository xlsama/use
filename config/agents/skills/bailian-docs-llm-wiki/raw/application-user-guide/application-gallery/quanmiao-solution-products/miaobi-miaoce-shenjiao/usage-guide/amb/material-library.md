# 素材库

在AI妙笔的素材库界面中，用户可以管理两种类型的素材：公共素材库和我的素材库。通过公共素材库和我的素材库，用户可以充分利用共享的素材资源，同时也可以建立自己的个性化素材库，提高创作效率和文章质量。

-   **公共素材库**
    

公共素材库是AI妙笔平台上同一技能组所共享的素材数据库。同一技能组下的用户可以在公共素材库中搜索和引用同一技能组下其他用户上传的素材内容。用户还可以将素材添加到公共素材库中，与同一技能组下的其他用户共享。这样可以为其他用户提供有价值的素材资源，同时也可以从其他用户分享的素材中获得灵感和创作参考。

公共素材库页面提供**新增素材**按钮、类型筛选下拉框和关键词搜索框，素材列表以表格形式展示各素材的**名称**、**上传人**和**最后修改时间**。

-   **我的素材库**
    

除了公共素材库，用户还可以创建和管理自己的素材库，即"我的素材库"。在我的素材库中，用户可以上传和保存个人搜集的各种素材内容。

在**我的**素材库页面，单击**新增素材**按钮，下拉菜单提供三种添加方式：**从本地上传**、**从外部链接添加**、**新增/粘贴纯文本**。素材列表展示素材名称、上传人和最后修改时间，并支持通过关键词搜索和分类筛选素材。
