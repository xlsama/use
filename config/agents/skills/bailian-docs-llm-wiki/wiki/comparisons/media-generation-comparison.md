# 图像生成、视频生成与 3D 生成对比

百炼平台提供了图像生成、视频生成和 3D 生成三大多媒体内容创作 API。三者均采用异步任务调用模式，但在输入输出格式、支持的模型系列、适用场景和计费方式上存在显著差异。本文从开发者技术选型的角度，对这三类能力进行系统对比。

## 核心维度对比

| 维度 | 图像生成 | 视频生成 | 3D 生成 |
|------|----------|----------|---------|
| **输入格式** | 文本提示词、参考图片、涂鸦草图、待编辑图片 | 文本提示词、首帧/尾帧图片、参考图片/视频/音频 | 文本提示词、单张图片、多视角图片（2-4 张） |
| **输出格式** | 图片（PNG/JPEG 等），支持多分辨率（最高 4K） | 视频文件，支持 480P-1080P 多种分辨率 | GLB 格式 3D 模型（含/不含贴图和 PBR 材质） |
| **主要模型系列** | 千问图像、万相（Wan/Wanx）、Z-Image、可灵 | 万相（Wan）、HappyHorse、可灵、PixVerse、Vidu | Tripo（H3.1 / P1.0） |
| **推荐模型** | `qwen-image-2.0-pro`、`wan2.7-image-pro`、`wan2.6-t2i` | `wan2.7-t2v`（文生视频）、`wan2.7-i2v`（图生视频） | `Tripo/Tripo-H3.1`（高精度）、`Tripo/Tripo-P1.0`（快速） |
| **API 端点** | `/services/aigc/multimodal-generation/generation` 或 `/services/aigc/text2image/image-synthesis` 等（因模型而异） | `/services/aigc/video-generation/video-synthesis`（主流）；部分旧模型使用 `/image2video/video-synthesis` | `/services/aigc/video-generation/3d-generation` |
| **调用模式** | 异步（创建任务 + 轮询结果） | 异步（创建任务 + 轮询结果） | 异步（创建任务 + 轮询结果） |
| **典型耗时** | 秒级至数十秒 | 1-5 分钟 | 数十秒至数分钟 |
| **支持地域** | 北京（全部）；部分模型支持新加坡 | 北京（全部）；万相 2.7、HappyHorse 支持多地域（新加坡、美国、德国） | 仅北京 |
| **SDK 支持** | HTTP、Python SDK、Java SDK | HTTP、Python SDK、Java SDK | HTTP、Python SDK、Java SDK |

## 功能覆盖对比

| 功能场景 | 图像生成 | 视频生成 | 3D 生成 |
|----------|----------|----------|---------|
| 文本生成内容 | 文生图 | 文生视频 | 文生 3D |
| 图片驱动生成 | 图生图、图像编辑 | 图生视频（首帧/首尾帧） | 单图/多图生 3D |
| 风格迁移 | 人像风格重绘、图像编辑 | 视频风格重绘（8 种预设风格） | 不支持 |
| 局部编辑 | 局部重绘、擦除补全、扩图 | 视频编辑（指令编辑、视频迁移） | 不支持 |
| 人像/人物 | 虚拟试衣、人物写真、表情包 | 数字人、人像动画、视频换人 | 不支持 |
| 文字处理 | 图中文字渲染、图像翻译、创意文字 | 不支持 | 不支持 |
| 多模态参考 | 多图参考生图 | 多模态参考生视频（图片+视频+音频） | 多视角图片输入 |
| PBR 材质 | 不适用 | 不适用 | 支持（可配置） |

## 模型生态丰富度

| 维度 | 图像生成 | 视频生成 | 3D 生成 |
|------|----------|----------|---------|
| 自研模型 | 千问图像、万相 | 万相 | 无 |
| 第三方模型 | Z-Image、可灵 | HappyHorse、可灵、PixVerse、Vidu | Tripo |
| 模型总数 | 20+ 个模型（含创意工具） | 15+ 个模型（含人像动画） | 2 个模型 |
| 版本迭代 | 万相 V1-V2.7 多代演进 | 万相 2.1-2.7 多代演进 | 单一版本 |

## 适用场景建议

### 图像生成

- **营销设计**：海报生成、商品图背景替换、创意文字排版
- **电商场景**：虚拟试衣、鞋靴试穿、虚拟模特换装
- **内容创作**：文生图配图、风格迁移、图像翻译
- **图像后处理**：扩图、超分、上色、擦除水印

### 视频生成

- **短视频制作**：文生视频快速生成短视频素材
- **数字人直播**：基于图片和音频生成数字人播报/唱演视频
- **视频二创**：风格重绘、视频编辑、视频换人
- **动画制作**：人像动画（舞蹈、表情包、口型同步）

### 3D 生成

- **游戏与元宇宙**：快速生成 3D 资产原型
- **电商展示**：商品 3D 模型生成与展示
- **工业设计**：概念验证阶段的快速 3D 建模
- **教育培训**：3D 教学素材快速生成

## 技术选型建议

1. **只需静态图片**：选择图像生成，模型成熟度最高，生成速度最快，场景覆盖最全面。推荐从千问图像或万相最新版入手。
2. **需要动态内容**：选择视频生成，支持文本、图片、音频等多种输入驱动，万相 2.7 为当前推荐版本。
3. **需要 3D 资产**：选择 3D 生成，目前仅 Tripo 系列可用，适合快速原型制作。如需高精度模型选 Tripo-H3.1（最高 200 万面），如需快速出结果选 Tripo-P1.0。
4. **多地域部署**：图像生成和视频生成的部分模型支持海外地域（新加坡、美国、德国），3D 生成目前仅限北京地域。
5. **成本考量**：三者均按任务计费。图像生成单次成本最低、耗时最短；视频生成成本和耗时适中；3D 生成需根据面数和贴图质量选择合适档位。
