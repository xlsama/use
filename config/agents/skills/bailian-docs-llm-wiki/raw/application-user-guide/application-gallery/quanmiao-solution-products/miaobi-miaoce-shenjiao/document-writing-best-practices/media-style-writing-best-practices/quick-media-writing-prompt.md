# 快速写一篇传媒稿（prompt一步式撰写）

文章主题（prompt）

请围绕数字经济释放全新增长动力写一篇新闻

## **具体操作**

1.  进入[妙笔](https://aimiaobi.console.aliyun.com/?product_code=g_broadscope_media&from=bailian#/home)首页，首先选择“传媒写作”。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7573555371/p894980.png)
    
2.  选择“直接生成”。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7573555371/p894985.png)
    
3.  按要求填写【主题】和【标题】内容，并且可以根据实际需求选择【文体】、【文章篇幅】、【输出语言】、【生成篇数】，完成后点击“开始写作”。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7573555371/p897484.png)
    
4.  等待系统创作内容。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7573555371/p897485.png)
    
    查看系统创建的内容是否符合预期：
    
    -   如果符合预期，点击“采纳”。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7573555371/p897487.png)
        
    -   如果对文章的引用来源不满意，可点击【结果干预】对引用源进行删改和【重新搜索】内容引用源功能，删改后可点击【重新生成】新文章。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7573555371/p897490.png)
        
    -   如对整体结果不满意，可使用【换一换】功能对文章内容进行【直接换】（重写）、【扩写】或【缩写】等操作。
        
        ![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7573555371/p897491.png)
        
    
    **说明**
    
    -   续写：根据选中原文继续延伸。
        
    -   扩写：根据选中原文改动词汇，扩大字数。
        
    -   缩写：根据选中原文改动词汇，缩减字数。

## **“搜索素材”丰富文章内容**

1.  点击“搜索素材”即可进入到素材搜索界面。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7573555371/p895038.png)
    
2.  在搜索框内输入您要搜索的内容即可，如：亚运会。
    
    您可以将符合您预期的素材内容加入素材库。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7573555371/p895045.png)
    
3.  在【素材库】找到您刚刚搜索到并添加的素材内容。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7573555371/p895047.png)
