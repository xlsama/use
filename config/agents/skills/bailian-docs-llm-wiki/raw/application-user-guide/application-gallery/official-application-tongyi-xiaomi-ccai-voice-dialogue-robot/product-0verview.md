# 语音对话机器人产品概述

本文档介绍通义晓蜜CCAI-语音对话机器人产品能力及产品优势。

## **什么是通义晓蜜CCAI-**语音对话机器人

通义晓蜜CCAI-语音对话机器人是一个支持快速搭建自定义音色的大模型机器人，支持大模型对话、语音播报、语音转写等能力，可实现语音大模型实时交互支持多端快速集成。

## **产品能力**

### Prompt构建模式

提供灵活的指令自定义能力，可通过结构化文本定义语音机器人的人设、角色、技能、约束等信息。同时支持语音的音色、音量、语速、语调等调节，让语音对话更加自然。

-   支持多种模型选择。
    
-   内置系统提供的指令模板。
    
-   自定义指令信息：实现与客户的标准对话服务；
    
-   语音配置包含选择大模型音色模板选择、音量、语速、音调配置。
    
-   高级配置中可以进行静默超时配置。

## **产品优势**

-   提供客户通过自定义Prompt的构建方式；
    
-   提供多种大模型音色的选择，同时支持对大模型音色。
    
-   支持语音方式对调试窗测试，配置完成后即可验证效果。
