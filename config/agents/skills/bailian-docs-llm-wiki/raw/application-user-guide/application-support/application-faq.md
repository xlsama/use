# 常见问题

您可以查阅有关阿里云百炼应用和知识库的常见问题。

## **应用中心**

1.  **目前官方都支持什么类型的插件？**
    
    目前系统提供六款插件能力，分别是：Python代码解释器、计算器、图片生成、夸克搜索、生成二维码、GitHub搜索。其中部分插件需要经过申请通过后才可以使用。
    
2.  **自定义插件能力是否会收费？**
    
    自定义插件服务目前暂时不收费，但配置智能体API时如涉及到prompt优化、应用调用及测试窗测试，则会收费。
    
3.  **对于我自己定义的插件或函数，****Assistant API****是如何理解插件或函数的参数，并使用它们的？**
    
    自定义API插件是可以理解的，会遵循协议，传给大模型，让大模型进行理解。对于函数而言，大模型会学习传入的参数信息，会返回完整的结果。
    
4.  **Agent****和****Assistant API****的最大区别是什么？**
    
    调整插件模型、基于上下文的理解，用户可以自己去开发，而Assistant API是可以提供各种类，方便调优。
    
5.  **知识检索增强（RAG）在哪些领域都得到了应用，展开说？**
    
    知识检索增强（RAG）模型凭借其强大的检索与生成能力，在问答系统、对话系统、文本摘要、知识图谱构建与推理、教育与培训、客户服务、新闻与内容创作以及智能搜索与推荐等多个领域均展现出广泛的应用潜力和价值。
    
6.  **在RAG检索增强应用进行测试时，会出现模型回复不准确的情况，应该如何优化？**
    
    您可以点击模型回复内容下方的问题反馈按钮，勾选问题类型并提交。也可以**复制RequestId**，通过提交阿里云工单反馈给我们。
    
    ![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5827845471/p948683.png)
    
7.  **AI输出后包含\*\*xxxxx\*\*，如何给他变成加粗显示？**
    
    \*\* \*\*是md格式的加粗标识。您可以在前端渲染大模型生成的内容时解析md语法并做对应的渲染。
    
8.  **应用的回复能设置成增量的么，当前每次都在前一次的基础上的全量回复。**
    
    stream=True, # 设置输出方式为流式输出；
    
    incremental\_output=True # 增量式流式输出；
    
9.  **RAG检索顺序是并行还是串行有先后顺序？**
    
    并行根据每个知识库的用户配置进行检索，然后再根据得分选取topN。
    
10.  **百炼调用自定义插件的时候能通过模型透传header吗，我们有个场景显示透传了header，但是根据服务端返回的情况，实际上并没有透传。**
     
     不支持自定义header， 仅支持authorization。
     

## **数据管理**

1.  **上传文件仅支持 pdf/doc/docx 文件, 请重试 错误码：140010**
    
    上传的文件为PDF格式时，请确保文件后缀为小写pdf。
    
2.  **每个业务空间最多上传10万个文档，客户的文件超过了应该怎么办呢，可以扩容吗？**
    
    提交阿里云工单进行申请。
    
3.  **上传文件接口必填的MD5参数，是什么作用？**
    
    用来验证上传的文件是否完整。
    
4.  **结构化数据中有100条数据，但导入后发现只有20条数据，是什么原因？**
    
    请确认表格中的数据中是否存在空行。产品策略中如果出现空行后面的数据将不会识别。如果第一行是空行，就会认为是空文件。
