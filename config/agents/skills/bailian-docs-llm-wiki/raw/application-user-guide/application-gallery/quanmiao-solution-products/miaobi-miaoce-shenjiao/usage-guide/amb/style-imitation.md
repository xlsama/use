# 文章风格和格式学习

## **功能概述**

AI妙笔提供了文章风格和格式学习的功能，可以帮助用户将文风类似的文件分析得到对应的文体模型，方便用户后续创作自定义文稿。文章风格和格式学习功能做到了为用户提供零代码训练新文本的能力，用户可以快速的操作上手。

## **功能入口**

[点击前往AI妙笔工作台](https://aimiaobi.console.aliyun.com)，点击以稿写稿。

![2](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4689678571/p1010214.webp)

## **具体步骤**

### **第一步：创建文体模型**

1.  在**以稿写稿**页面，上传本地素材或从素材库添加文章。
    
    -   点击**上传本地素材**，可以选择本地的文件进行上传。
        
    -   点击**从素材库添加**，支持从**素材库**进行搜索素材添加，也支持添加**自定义文本**。
        
        ![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4689678571/p1010220.png)
        
    
    下图为已添加素材文章的页面示例：![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4689678571/p1010223.png)
    
    **说明**
    
    -   要求至少上传一篇文章，且总字数在50000字以内；
        
    -   冷门、制式、强格式的表格类型文章均不可学习模仿，用户上传的文章需要对其标准有一定的说明。
        
    
2.  点击**开始分析**，进行素材文章分析，分析完成后页面示例如下。
    
    > 文章素材分析过程中，支持手动**终止生成**。
    
    ![3](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4689678571/p1010228.webp)
    
    -   点击**重新生成**，模型可以对上传文章素材进行重新分析总结。
        
    -   点击**手动调整结果**，可以对模型生成内容进行手动编辑修改。
        
        ![4](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4689678571/p1010230.webp)
        
    -   点击**弃用**，可以直接**弃用**原始**所有**模型分析结果。
        
3.  当用户采纳当前模型的分析结果，点击**采纳并命名文体**，填写文体风格名称，生成对应的文体风格模块。
    
    ![5](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4689678571/p1010234.webp)
    

### **第二步：使用文体**

1.  点击**使用文体**，用户可以开始创作文稿。
    
    ![7](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4689678571/p1010241.webp)
    
2.  [创作文稿](https://aimiaobi.console.aliyun.com//styles#/create)页面如下图所示，可以在**文体**页面中选择所需文体。
    
    ![8](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4689678571/p1010247.webp)
    

## **扩展功能**

### **文章风格和格式学习增强效果**

点击**增强效果**，可重新编辑并分析文章素材，强化已有文体。

![10](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4689678571/p1010265.webp)

### **调研反馈**

点击**提交反馈**，可以填写对当前文体模型的**期望的效果**和**改进建议**。

![9](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4689678571/p1010261.webp)
