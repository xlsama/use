# CCAI如何进行集成；查看技术集成方案

## **技术集成方案**

提供CCAI产品的集成方案：

-   简单介绍了CCAI产品集成时的需要的前置条件。
    
-   简单介绍了企业的服务平台方式进行业务对接。
    
-   简单介绍了在企业服务平台拿到不同的对话数据通过对应OpenAPI上传到智能对话分析服务，以实现服务质检能力，再将其结果返回到对应控制台中。
    

在实际集成中遇到问题时，可添加服务钉钉群（78410016550）进行咨询。
