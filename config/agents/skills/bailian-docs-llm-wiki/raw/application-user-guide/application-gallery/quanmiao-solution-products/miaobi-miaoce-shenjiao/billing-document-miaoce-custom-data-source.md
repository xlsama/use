# 计费说明（妙策-自定义数据源）

本文档主要介绍妙策-自定义数据源接口的计费方式。

## **计费说明**

-   计费方式：全妙系列产品中的妙策-自定义数据源API按实际消耗的Token后付费，仅开通服务不收费。
    
-   计费逻辑：通过全妙API接口调用模型的输入和输出的总Token数计费。
    
-   平台提供以下2种按使用Token后付费的模型规格：
    
    -   全妙-Plus 0.002元/千Token；
        
    -   全妙-Max 0.06元/千Token。
