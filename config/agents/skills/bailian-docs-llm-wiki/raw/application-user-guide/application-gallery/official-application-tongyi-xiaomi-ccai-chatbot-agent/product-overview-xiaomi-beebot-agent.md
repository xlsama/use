# 产品概述

下面的文档是关于通义晓蜜对话Agent的产品概述。

## **1\. 产品介绍**

通义晓蜜对话Agent是快速构建企业客服机器人的 LLM 原生应用， 基于晓蜜客服场景专属大模型，帮助企业开发者添加业务流程编排快速构建专属的客服场景对话机器人。

## **3\. 核心产品能力**

### **3.1** 对话Agent构建

提供客服 Agent 的构建，支持接入业务对话流程、高频问答知识与用户进行对话：

-   内置通用的人设编排模板；
    
-   对话流程技能：实现与客户的标准对话服务；
    
-   用户高频率提问的问题以及它对应答案，可以维护在高频问答知识中；
    
-   更多设置包含模型参数调配、通用知识、回复语言设置、敏感安全策略、服务兜底等客服通用能力。
    

对话Agent配置页面左侧可设置**角色设定**（姓名、职责）、**限制条件**、**机器人开场白**，并在**知识库**区域绑定流程技能（如"查天气"）。右侧**对话测试**面板支持开启调试模式，可实时验证流程技能的触发与应答效果。

### **3.2 可视化流程技能编排**

场景化业务流程编排：

-   提供回复、判断、API 插件、参数收集节点
    
-   回复节点支持固定回复和 prompt 指令两种模式
    
-   参数收集、API 插件提供结构化信息收集和调用，与外部系统进行二次集成
    

以**查天气**工作流为例，**开始**节点通过意图分支连接两条路径：一条经**意图1**直接调用插件并由**机器人回复**节点输出结果，另一条经**意图2**先进入**参数收集**节点完成参数收集后再调用插件。

用户在对话测试页面输入**查天气**，机器人通过参数收集节点追问**请问您想查询哪个城市的天气呢？**，答案来源显示匹配到**查天气**流程技能。

### **3.3 高频问答知识配置**

用户高频率提问的问题以及它对应答案，可以维护在高频问答知识中，用于用户的知识问答。也可通过设置高频问答知识，对文档中难以回答的问题进行干预解决。

在**FAQ编辑**弹窗中，选择**FAQ配置**页签，填写**问题**（例如"对话机器人在哪里登陆"），选择**答案类型**（**纯文本**或**富文本**），在**答案**富文本编辑器中输入答案内容（支持添加超链接等格式），设置**生效时间**（如**永久生效**），然后单击**提交**。提交后该FAQ条目将显示在知识库列表中，状态为**生效中**。

在测试对话中，用户发送问题**智能对话机器人在哪里登陆**，机器人基于高频问答知识生成回答**登陆地址：智能对话机器人控制台**（含超链接），并显示**生成完成**状态及答案来源引用。

## **4\. 产品优势**

-   基于Qwen模型，围绕客服咨询服务场景专属训练的晓蜜大模型；
    
-   提供客户Agent的构建方式，支持低成本接入业务对话流程，能支持与用户进行对话；
    
-   可控流程技能，严格遵循流程技能与用户进行服务，支持系统集成。
