# 官方应用-析言GBI

析言GBI是基于阿里云通义大语言模型在数据分析领域专门增强的原生数据助理，通过自然语言交互实现NL2SQL、数据问答、分析、洞察等多维度的大模型智能分析应用，适用于业务变化较快、数据分析时效性强的一线业务场景。
