# 妙笔首页概览

AI妙笔首页的相关说明。

## **文体写作工具**

AI妙笔首页界面展示五种文章创作类型，具体内容如下：[点击前往AI妙笔工作台](https://aimiaobi.console.aliyun.com)

-   传媒写作场景：为新闻机构、媒体公司、广告代理商等机构提供高质量的新闻稿件等内容。
    
-   公文写作场景： 为各级政务机关的公务员撰写常见公文， 比如工作报告、领导讲话、复函、通知公告等。
    
-   营销写作场景： 为品牌、零售商、电子商务平台等提供创意丰富的营销文案，包括宣传册页、产品描述、广告语、市场推广计划和社交媒体内容等。
    
-   办公写作场景：为多种团队和组织，旨在促进工作流程的顺畅、信息的有效传递及团队成员间的协作，包括学习心得、周报等。
    
-   自定义写作场景：为其他组织团体提供符合他们需求的文稿服务。
    

### **查看**全部文体

点击**全部**，即可查看传媒、公文、营销、办公场景的文体类型。

-   **传媒类文体**：新闻评论、新闻写作、新闻简讯、报纸文章、解说词、网友评论、通用通知、公关稿口号、朋友圈文案、演讲稿、视频镜头脚本、短诗歌、漫画脚本、小说等。
    
-   **公文类文体**：工作报告、征求意见、通知、通告、通报、请示、决定、函、决议、公告、政府意见信、公示、公报、命令、批复、议案等。
    
-   **营销类文体**：广告推广、营销文案、视频分镜脚本、小红书文案等。
    
-   **办公类文体**：日周月报、学习心得等。
    

## **灵感创作工具**

-   文章风格和格式学习：可以帮助用户将文风类似的文件分析得到对应的文体模型，方便用户后续创作自定义文稿。文章风格和格式学习功能做到了为用户提供零代码训练新文本的能力，用户可以快速的操作上手。
    
-   广告生成：通过用户输入产品的相关信息，大模型即可生成对应的广告文案。主要分成三步走，用户输入产品主题和相关的信息，让大模型可以更好的学习，生成更加适配的广告文案。
    
-   一键成片：基于口播脚本和素材批量混剪视频，面向广电传媒、金融、泛娱乐内容平台、营销等多行业领域，让视频生产力不断提速，内容生产成本不断降低，通过全智能带动视频创作颠覆式提效。一键成片功能有以下优势：
    
    全智能提效视频生成：基于用户输入的脚本（支持AI生成脚本）和上传的视频素材，运用大模型能力进行智能化分析，依据分析结果自动匹配素材关联片段，并通过AI音色合成、片段优选、自动对齐、字幕背景智能组合等操作，一键实现视频内容生成的全链路、全智能提效。
    
    多维场景：基于深厚的剪辑制作服务能力积淀，在成片风格、时长、分辨率尺寸等维度，具备高自由度的个性化定制能力及扩展性，满足不同场景应用下的多样化需求。
    

在首页主内容区，五个写作分类卡片（**传媒写作**、**公文写作**、**营销写作**、**办公写作**、**自定义写作**）下方展示文体标签按钮（如函、解说词、短诗歌、公示、通用文体等），单击右侧**全部**可查看全部文体。

## **搜索素材**

搜索素材：可以通过输入问题来搜索相关素材信息，用于创作文章。

**搜索素材**按钮位于首页右上角。
