# 文本生成模型API参考

百炼提供多种接口用于调用文本生成模型：

-   [OpenAI 兼容 Chat Completions](https://help.aliyun.com/zh/model-studio/qwen-api-via-openai-chat-completions)：与 OpenAI 客户端库直接兼容，迁移现有应用或接入第三方工具的成本最低。
    
-   [OpenAI兼容-Responses](https://help.aliyun.com/zh/model-studio/openai-compatible-responses/)：内置联网搜索、代码解释器和网页内容提取工具，自动管理对话历史，无需手动维护。
    
-   [Anthropic兼容-Messages](https://help.aliyun.com/zh/model-studio/anthropic-api-messages)：兼容 Anthropic Messages API，支持思考和工具调用。
    
-   [DashScope](https://help.aliyun.com/zh/model-studio/qwen-api-via-dashscope)：百炼原生接口，提供最完整的功能集和参数支持。
