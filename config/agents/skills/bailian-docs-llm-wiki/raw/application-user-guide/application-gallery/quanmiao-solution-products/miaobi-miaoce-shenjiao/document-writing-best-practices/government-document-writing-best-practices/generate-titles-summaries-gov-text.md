# 用已有文章，生成标题摘要等

文章来源

本地已经编辑好的文章，如word文件

需求点

对此文章生成全文或者局部 “生成标题/摘要/续写/扩写/缩写”；或者“ 改变文体/文风”

## 具体操作

### 第一步：文章准备

-   首先，准备好本地文章内容
    

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0781943071/p750252.png)

-   进入到AI妙笔系统创作界面，选择“创作政务公文”
    

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0781943071/p750244.png)

-   复制刚刚准备好的文章内容到编辑界面
    

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0781943071/p750246.png)

### 第二步：生成

#### 1).生成全文标题

**说明**

随后即可根据您的需求在【AI工具箱】下进行生成标题、摘要、改写文体文风等操作；由于生成全文续写、摘要与生成全文标题操作方式一致，下面以“生成全文标题”作为操作演示

-   点击“生成全文标题”即可
    

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0781943071/p750243.png)

-   选择一个合适的标题，随后点击“采纳并编辑”即可
    

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0781943071/p750251.png)

-   生成的标题效果如下：
    

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0781943071/p750248.png)

#### 2).生成局部标题

**说明**

由于生成局部摘要、续写、扩写、缩写与生成局部标题操作方式类似，下面以“生成标题”作为操作演示

-   首先，鼠标拖拽到您需要生成标题的局部文章段落中，随后会弹出下图箭头所示的小弹窗
    

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0781943071/p750250.png)

-   随后鼠标点击此按钮，会弹出下方“快捷生成”浮窗，包括：生成标题、摘要、续写、扩写、缩写等操作。
    

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0781943071/p750249.png)

**说明**

续写：根据选中原文继续延伸 扩写：根据选中原文改动词汇，扩大字数 缩写：根据选中原文改动词汇，缩减字数

-   点击“生成标题”即可对此局部内容生成标题，选择您需要的标题，随后点击“采纳”
    

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0781943071/p750247.png)

-   生成的标题效果如下：
    

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0781943071/p750245.png)
