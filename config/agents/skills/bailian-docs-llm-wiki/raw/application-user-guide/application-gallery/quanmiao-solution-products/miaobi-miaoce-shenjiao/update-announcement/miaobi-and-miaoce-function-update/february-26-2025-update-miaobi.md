# 2025年2月26日更新-妙笔

### **迭代说明**

本次迭代妙笔全量化升级，引入Deepseek模型，整体优化了交互界面，写作效果得到提升，写作速度提升，使用门槛进一步降低。

产品地址：[妙笔产品控制台](https://aimiaobi.console.aliyun.com/?product_code=g_broadscope_media&from=bailian#/home)。

升级后的妙笔首页定位为"深度服务媒体行业的专业文体写作工具"，提供**传媒**、**法定公文**、**事务公文**、**营销**、**自定义**五大写作类别，并集成**AI搜索素材**、**AI选题策划**、**文章风格和格式学习**、**一键成片**四项 AI 辅助功能。左侧导航栏包含**首页**、**内容管理**、**素材库**等入口。

### **具体更新细节**

#### **1\. 妙笔接入Deepseek，且写作时支持切换模型**

传媒类-通用文体、法定公文类、事务公文类和营销类写作支持选择模型，新增Deepseek-r1模型可选。

在**AI起稿**面板的**写作模型**下拉框中，可选择**deepseek-r1**或**妙笔写作模型**。

#### **2\. 每种文体最多支持3种写作方式**

-   直接写作：所有文体都支持，即输入相关信息后可直接生成文章；
    
-   分步骤写作：即按照输入主题->AI写大纲->AI摘编素材->写文章，目前传媒类中的新闻评论和新闻报道支持；**分步骤写作**：部分文体（如新闻报道）支持分步骤写作，流程为输入主题后由AI生成大纲并指定素材，再由AI填编素材，最终生成文章。选择对应文体后，在弹出的**选择写作方式**对话框中选中**分步骤写作**并单击**确认**即可。
    
-   高级模式：所有文体都支持高级模式写作，即输入写作主题和指令（prompt）的方式写文章，使写作更加灵活自由。
    
    在高级指令面板中，**其他指令（prompt）**文本域支持以##角色、##技能、##限制等结构化格式编写指令（限2000字）；**写作参考素材**区域可开启**通过互联网搜索来补充素材**开关以提升写作质量和准确性，也可通过**自定义素材**手动添加素材（最多20个）；**文章篇幅**下拉框可设置文章长度限制。
    

#### **3\. 新增写作文体以及重新分类**

1.  文体重新分类，将原先“公文类”和“办公类”文体重新整合分为“法定公文类”和“事务公文类”，并新增9个专业的文体，整体文体以及分类如下：
    
    1.  法定公文：即国务院规定的15种文体，分别是报告、通告、通报、请示、决定、函、决议、公告、公报、命令、批复、议案、意见、纪要、通知；
        
    2.  事务公文：公示、工作汇报、调研报告、意见建议、演讲稿、接待方案、项目书、征求意见、政府意见信、日周月报、学习心得、通用文体。
        
        传媒类：新闻评论、新闻报道、通用文体。营销：广告词生成、小红书笔记、商品测评文案、朋友圈文案、微信公众号文案、营销宣传文案、视频分镜脚本、通用文体。自定义：用户可自行创建和管理个性化写作文体。
        

#### **4\. 公文类**文体新增模板输入

能保证对应文体生成的正文都包含关键信息，使输出结果更加专业。公文类文体中选择**事务公文类 / 公示**模板后，进入**AI起稿**页面，表单包含以下字段：**标题**（示例"关于XX拟晋升人员名单公示"，限1000字）、**缘由**（限1000字）、**事项**（限1000字）、**公示时间**（限200字）、**结语**（如"特此公示"，限500字）、**主送机关**（限200字）、**发文机关**（限500字）、**发文时间**（日期选择器）和**写作模型**（下拉框）。填写完成后单击底部**直接写文章**按钮生成公文，也可单击**高级模式**进行更多配置。

#### **5\. 支持长文写作**

最多可支持5000字以上的长文的输出，比如演讲稿。

在字数限制的下拉框中选择**5000字左右**。

#### **6\. 优化正文输出格式**

-   根据输入的内容模型可生成文章标题；
    
-   优化正文输出格式，将原始的Markdown格式渲染成大小标题。
    

优化后的正文呈现多级标题层次：文章主标题（如**城市社区养老服务：现状、挑战与对策**）、章节标题（如**现状分析**、**问题与挑战**）以及编号子标题（如**1\. 财政资金匮乏**）分别以不同字号和样式渲染显示。

#### **7\. 优化配图策略**

-   事实类场景，推荐使用“搜索图片”和“来自素材库”；
    
-   创意类场景，推荐使用“AI智能配”。
    

在左侧导航栏选择**配图**，单击**AI智能配图**页签进入配置面板。选择生成模型（**万相**、**万相lite**或**FLUX**），在文本输入区输入图片内容或主题描述（限500字），选择风格（**默认**、**水彩**、**油画**、**中国画**、**扁平插画**、**二次元**、**素描**、**3D卡通**），并设置图片尺寸（如**1:1 (1024\*1024)**）。

#### **8\. 新增辅助写作提速的工具**

1.  用户可收藏模板提示词，便于后续应用。在写作工具界面中，输入框支持最多1000字符的文本输入，可通过**写作模型**下拉框选择模型，单击**☆ 收藏以上输入的内容**按钮可将当前输入内容收藏以便复用。
    
    单击页面顶部的**收藏**按钮，弹出**我收藏的提示词**浮层，展示已保存的模板列表及搜索框。找到目标模板后，单击对应的**一键填入**按钮，即可将模板内容自动填入表单各字段。
    
2.  法定公文和事务公文类文体提供一键填入示例，帮助用户如何填写输入。
    
    单击**一键填入示例**后将覆盖已填入的模板参数，请谨慎操作。
    
3.  将写作的常用功能模块以菜单的形式展示，高效处理写作任务。
    
    左侧导航栏包含**文体**、**AI起稿**、**素材**、**配图**、**AI工具**五个功能入口。单击**文体**后，可按分类（传媒类、法定公文类、事务公文类、营销类、自定义类）选择具体文体模板。
