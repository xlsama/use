# 千问联网检索Agent产品简介

千问联网检索Agent，可实时检索全网多维多源信息，自动筛选权威来源并提炼要点，支持追溯引用与多轮追问，快速生成可靠答案，提升决策与效率。

## **产品体验**

### **应用开通**

1.在阿里云百炼控制台的应用广场中点击[千问联网检索Agent](https://bailian.console.aliyun.com/cn-beijing?spm=5176.28197581.0.0.3d7b74a1gf57op&tab=app#/app/app-market/web-search-agent)卡片，进入应用详情。

2.首次试用时，点击右上角**前往开通**完成应用开通。

### **应用管理**

点击**我的应用**进入应用管理页面，页面展示所有已创建的应用和应用key等信息，首次使用需要新增应用。

### **应用配置**

点击应用卡片或新增应用进入应用配置界面。

#### **1.基础信息**

-   **应用简介**：概述应用定位与核心能力，供用户参考，必填，限制200字符
    
-   **开场白**：首次进入对话时，系统触发的第一条响应消息，非必填，限制500字符
    

#### **2.检索配置**

-   **场景增强：**支持多场景自由勾选与联动，实现叠加多种增强效果
    
    -   **本地生活：**开启后，系统将支持对当前位置进行周边酒店、餐饮及公共设施等精准检索，当与“图文并茂”同时打开，可增加附带图片的 POI卡片信息展示
        
-   **智能改写**：模型对原始query进行标准化重构，可以优化检索匹配度，建议开启
    
-   **垂域工具**：配置垂域工具，模型会依据query上下文自动锁定最佳检索源，可实现即时性与高精度召回双重提升
    
-   **信息时效性范围限制**：支持自定义时间跨度，实现时间区间定向检索功能
    
-   **指定检索范围**：开启后，单击**配置**，在弹出的**指定检索范围**对话框中单击**\+ 添加网站**，输入目标网站域名（最多100个），例如`tieba.baidu.com`、`baike.baidu.com`，单击**确定**完成配置，系统将优先从此范围网站获取信息，若无匹配信息则会切换至其他网站搜索，实现"精准优先 + 全域保障"的双重覆盖机制
    
-   **单轮检索条数：**数量越高信息越全，但会占用更多资源，增加耗时
    
-   **网页深度阅读：**开启后将提升检索效果，但会增加耗时
    

#### **3.策略配置**

-   **自动判定**：由模型自行判断是否需要联网构建回复
    
-   **边想边搜模式：**模型会根据query难易程度自动决策检索范围及轮次
    
-   **极速模式：**系统以最小化响应时间（RT）为核心，适用于高并发、低延迟需求，建议仅在对耗时敏感、对召回精度容忍度较高的场景中启用
    
-   **自定义干预搜索的触发策略：**支持自然语言精确控制检索范围（如特定主题/地域），当查询命中预设的检索范围时，会触发外部数据检索，若判定为非相关领域，系统将直接调用模型自身的通用知识进行响应
    
-   **强制检索：**所有查询均自动触发联网检索
    

#### **4.生成配置**

-   **system prompt**：检索及生成时会根据填入信息执行约束策略，限制1万字符
    
-   **百度词条下划线：**开启后，模型会根据生成结果，自动解析语义结构并优选代表性词条，无需人工干预
    
-   **图文并茂：**开启后，在生成结果中会输出文字及穿插上下文关联图像，实现图文深度融合的效果
    
-   **相关视频：**开启后，在生成结果中均会附加与上下文相关的视频素材，并统一置于内容底部，构建沉浸式多模态回答
    
-   **追问：**开启后，在生成结果后均有1-3个相关问题提示，选择提示问题后会参考历史对话内容及system prompt信息进行回答
    
-   **引用：**开启后，系统将在生成内容中涉及检索来源的段落处，自动增加数字角标，实现信息来源的可视化追溯
    
-   **设置角标样式：**支持根据需求灵活配置角标的外观样式
    

### **应用测试**

配置完成后，可在输入框输入query进行测试，测试结果将展示检索回答内容、参考来源链接、token 统计信息（输入/输出/用时）及推荐追问问题。

-   **附件上传**：
    
    数量限制：单次最多支持上传 10 张图片
    
    文件大小：单张上限 4MB
    
    支持格式：JPG/JPEG、PNG、WebP、BMP、JPE
    
-   **位置信息**：系统支持自动定位/手动录入双重位置获取模式，在此基础上，提供从身边便民搜索到全城吃喝玩乐指南，再到一站式的旅行规划服务，全方位满足多样化的出行与生活需求。
