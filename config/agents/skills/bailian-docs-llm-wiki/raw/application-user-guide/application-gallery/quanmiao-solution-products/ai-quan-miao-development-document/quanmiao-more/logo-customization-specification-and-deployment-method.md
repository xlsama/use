# 全妙Logo定制规范及部署方式

本文档是关于全妙SaaS产品Logo定制规范和部署方式的介绍。

## **Logo可定制范围**

在登录页、菜单栏支持定制您企业内部统一Logo。

1.  logo容器元素特征：
    
    1.  登录页logo容器元素无特别；
        
    
    ![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4326577671/p1041365.png)
    
2.  菜单logo容器元素：整体要求是水平左对齐，垂直居中对齐，logo高度铺满。
    
    1.  容器尺寸：
        
        1.  无产品切换按钮时，可支持宽度190px，高度32px ；![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4326577671/p1041363.png)
            
3.  当对接全妙多个产品时（妙笔、妙搜、妙策），会有切换按钮，logo容器可支持宽度140px，高度32px ；![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4326577671/p1041364.png)
    
4.  标签页logo（仅私有化部署支持修改）：设计规范：32\*32px，背景透明。
    
    ![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2512283771/p1059698.png)
    

## **设计规范**

1.  内容规范：背景透明，图文一体；
    
2.  尺寸规范：您可综合考虑区域尺寸及示例效果，设计您的logo尺寸（宽高比参考logo容器元素，2倍图最优）。
    

### **示例**

-   示例1：宽高比是140:32（上下无留白），效果如下：![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4326577671/p1041370.png)![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4326577671/p1041369.png)
    
-   示例2（妙笔原logo）：宽高比134×32（图片内上下各有留白6px，图片内非空白内容高度是20px），效果如下： ![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4326577671/p1041368.png)![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/4326577671/p1041367.png)
    

## **部署方式**

您可自主设计 Logo，并通过 [iframe嵌入方案](https://help.aliyun.com/zh/model-studio/iframe-embedding-scheme)完成部署。系统将为您生成专属的公有云 iframe 访问链接，通过iframe嵌入方案进行定制。
