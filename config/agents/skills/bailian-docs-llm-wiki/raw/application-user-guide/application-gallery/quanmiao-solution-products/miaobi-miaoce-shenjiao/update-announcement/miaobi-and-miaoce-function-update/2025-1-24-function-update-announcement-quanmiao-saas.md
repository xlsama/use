# 2025年1月24日更新-全妙解决方案类产品

2025年1月24日全妙解决方案类产品功能更新公告。

## **妙笔**

**说明**

产品地址：[妙笔产品地址](https://aimiaobi.console.aliyun.com/?product_code=g_broadscope_media&from=bailian#/home)。

### **1\. 写作链路优化**

功能说明：

-   通用文体链路优化，跳过摘编节约大模型推理次数；在**选择文体**弹窗中，文体分为**传媒类文体**、**公文类文体**、**营销类文体**、**办公类文体**四个类别，每个类别下均包含**通用文体**选项。
    
-   传媒场景直接写作更新模型 ；
    
    在**直接写作**页签中，填写**写作内容或主题**（必填，上限2000字）并选择**文体**（如**新闻类 / 新闻写作**），即可基于更新后的模型生成文章。
    
-   全文续写和选中文本续写功能更换为最新模型。
    
    该功能入口位于编辑器顶部的**AI工具箱**下拉菜单中。
    
    在文档编辑器中，选中文本后右键打开快捷菜单，选择**续写**即可调用该功能对选中内容进行续写。
    

### **2\. 文章风格和格式学习**

功能说明：升级应用的学习和写作链路，提升首包响应时间，通过改进算法和数据处理方式，大幅减少创作消耗大模型的消耗的推理额度。​

在写作工具主页面的**灵感创作工具**区域，单击**文章风格和格式学习**卡片进入该功能。

### **3\.** 生成文章标题减少消耗的推理额度

功能说明：减少生成文章标题消耗的大模型推理次数。

AI妙笔生成文章后，编辑器正文区域顶部显示生成的文章标题（如**西湖导游新闻：传递文化与关怀**），页面底部显示生成来源及字数统计。
