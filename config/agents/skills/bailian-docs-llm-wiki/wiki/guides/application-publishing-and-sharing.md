# application publishing and sharing

阿里云百炼平台支持将已创建的智能体应用和工作流应用通过多种渠道发布和分享，包括网页链接、钉钉机器人、微信公众号、组件复用以及音视频实时互动。此外，通过 UI 设计器可以低代码搭建自定义应用界面并发布为网页 UI 应用。这些能力帮助开发者将 AI 应用快速集成到实际业务场景中。

## 发布渠道概览

百炼应用支持以下发布方式：

| 渠道 | 适用应用类型 | 说明 |
|------|-------------|------|
| 官方网页版 | 智能体应用 | 生成分享链接，默认有效期 7 天，可调整 |
| UI 应用（魔笔） | 智能体/工作流 | 通过 UI 设计器搭建自定义界面后发布 |
| 钉钉机器人 | 智能体应用 | 需创建钉钉应用并配置回调地址 |
| 微信公众号 | 智能体应用 | 需微信公众号 AppID 授权 |
| 组件 | 智能体/工作流 | 发布为可复用模块，供其他应用调用 |
| 音视频实时互动 | 智能体/工作流 | 语音/视频互动，支持 H5 扫码和 SDK 集成 |

详细操作流程参见 [分享智能体应用](../../raw/application-user-guide/application-publishing-and-sharing/share-an-application.md)。

## 官方网页版分享

1. 在**应用管理**页面点击目标应用的**发布**，选择**官方网页版**并点击**创建**。
2. 发布后页面显示分享链接及失效时间，可通过下拉框调整有效期。
3. 复制链接分享给目标用户，持有链接的阿里云用户均可访问。
4. 点击**网页下线**可随时停止服务。

**权限与计费**：应用创建者（主账号）、RAM 用户和持有链接的阿里云用户均可访问。所有通过分享链接产生的费用由应用创建者 UID 账号承担。

可选开启**应用观测**以统计网页版应用消耗的 Token 数量。

## 钉钉机器人集成

集成钉钉需要完成以下步骤：

1. **授权**：首次使用需授权计算巢 AppFlow 提供自动化集成服务（SLR 关联 + API-KEY 加密传输）。
2. **创建钉钉应用**：在[钉钉开放平台](https://open-dev.dingtalk.com/)创建应用，获取 Client ID 和 Client Secret。
3. **创建消息卡片**：在卡片平台创建 AI 卡片模板，获取模板 ID。卡片类型选择**消息卡片**，场景选择**AI卡片**。
4. **授予权限**：为应用申请 `Card.Streaming.Write` 和 `Card.Instance.Write` 两个权限。
5. **配置百炼**：在百炼发布渠道填入模板 ID、Client ID、Client Secret，获取回调地址。
6. **配置机器人**：在钉钉应用中添加机器人能力，消息接收模式选择 **HTTP 模式**（Stream 模式不支持），填入回调地址后发布。

> **注意**：消息接收模式必须选择 HTTP 模式，选择 Stream 模式会导致无法返回消息。

完整配置流程参见 [分享智能体应用](../../raw/application-user-guide/application-publishing-and-sharing/share-an-application.md)。

## 微信公众号集成

1. 在发布平台选择**微信公众号**，选择 API KEY 后进入配置。
2. 需要提供微信凭证（开发者 AppID），可在[微信公众号后台](https://mp.weixin.qq.com/)的**设置与开发 > 开发接口管理 > 基本配置**中获取。
3. 配置完成后生成微信客服二维码，扫码即可体验 AI 机器人。

## 发布应用为组件

组件是百炼平台的复用机制，允许将智能体或工作流应用封装为模块化组件，供其他应用接入调用。详细的组件开发和使用方法参见 [使用智能体或工作流作为组件](../../raw/application-user-guide/application-publishing-and-sharing/use-agent-or-workflow-as-component.md)。

### 组件配置要点

- **组件名称和描述**：描述会影响智能体自动调用的判断逻辑（在工作流中仅作说明用途）。
- **系统参数**：预设 `query`（String，必填）和 `imageList`（Array\<String\>，非必填），不可删除但可隐藏。
- **传参方式**：
  - **业务透传**：由调用者或上游节点提供参数值。
  - **模型识别**：由大模型根据参数描述自动推断填充。在工作流中即使设为模型识别，也必须显式提供参数值。

### 使用限制

- **避免嵌套调用**：A 调用 B、B 调用 A 会导致循环调用，功能不可用。
- **避免多级调用**：A 调用 B、B 调用 C 容易因运行时间限制产生超时错误。
- **自动更新**：应用重新发布后，对应的组件也会自动更新。

## UI 设计器

UI 设计器集成了阿里云魔笔低代码平台能力，支持通过拖放组件构建应用界面。详细操作参见 [UI设计器](../../raw/application-user-guide/application-publishing-and-sharing/ui-designer.md)。

### 核心能力

- **低代码搭建**：可视化编辑器，拖放组件构建页面。
- **服务集成**：支持接入百炼智能体、大模型、数据库和 HTTP 服务。
- **权限管理**：内置测试账号体系，支持钉钉/企业微信一键登录，兼容 OIDC/OAuth 2.0。
- **模板支持**：提供智能出行助手、智能体门户、AI 基础对话、企业 AI 知识库 Lite 四种预置模板。

### 发布环境对比

| 项目 | 开发环境 | 生产环境 |
|------|---------|---------|
| 用途 | 开发、调试和验证 | 终端用户实际使用 |
| 有效期 | 24 小时后失效，需重新发布 | 长期有效 |
| 费用 | 免费 | 需订阅付费套餐 |
| 域名 | 平台提供 | 支持自定义域名 |

### 访问权限配置

发布后默认持有链接的阿里云用户可访问。可通过登录配置开启**允许匿名访问**，并通过权限组控制匿名用户可访问的页面范围。

### 计费说明

- UI 设计器本身不计费。
- AI 对话产生模型调用费用。
- 文件存储免费额度 1GB，数据库免费额度 0.3GB，超出按量计费。
- 生产环境发布需订阅团队版（或更高级别）套餐。

## 音视频实时互动

支持图文对话类应用（智能体和工作流）通过语音或视频方式实时互动：

1. 在应用的 **AI 实时互动**页签配置语音/视频互动，选择 API KEY。
2. 生成临时体验二维码（有效期 24 小时），通过手机扫码体验。
3. 发布后支持两种渠道：
   - **H5/APP 扫码**：生成分享链接，设置 Token 有效时间。
   - **SDK 集成**：支持快速集成（WEB/iOS/安卓）和基于 AICallKit SDK 的开发集成。

> **注意**：百炼应用、API Key 和 UI 设计需要归属于同一[业务空间](../concepts/workspace.md)，否则无法正常关联使用。
