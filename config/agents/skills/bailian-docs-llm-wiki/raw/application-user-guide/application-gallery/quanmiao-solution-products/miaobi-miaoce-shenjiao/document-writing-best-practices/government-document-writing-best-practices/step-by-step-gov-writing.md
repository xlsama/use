# 分步式撰写政务稿（精准控制章节内容）

文章主题（prompt）

请以政府的口吻，帮我写一篇通报，篇幅为中篇，标题为：达州市疫情防控最新通报

文章标题

达州市疫情防控最新通报

关键词

疫情、防控、核酸、通告

## 具体操作流程

-   首先选择“创作政务公文”
    

该选项位于首页**创作文章**功能卡片展开的子选项中。

-   按系统要求填写【写作内容或主题】，输入【文章标题】、【写作场景】、【关键词】后点【下一步】
    

-   查看生成的大纲
    

1).全部换一换

如果对生成的大纲不满意，只需点击“重新生成”即可生成新的大纲。

2).编辑大纲

如果对生成的大纲相对满意，但是需要修改细节，那么只需点击各大纲条目右侧的编辑图标，点击后大纲内容进入可编辑状态，可直接修改标题及子项内容。

点击引用按钮图标可以添加外部内容或文章链接进行引用。

在弹出的**自定义素材**弹窗中，通过**素材库**或**搜索素材**页签查找素材，素材库下可切换**公共**和**我的**分类，单击素材右侧**\+ 添加**按钮选择所需素材，再单击**确认**完成引用。

-   查看生成的摘编
    

如果对生成的摘编不满意，只需点击“重新生成”即可进行优化操作。

-   最后，点击“保存”即可保存此文章至【文档管理】中。
    

## 素材智搜丰富文章内容

-   在AI妙笔首页，单击**搜索素材**功能卡片进入素材搜索页面。
    
-   随后在搜索框内输入您要搜索的内容即可，如：关于搬迁老城区儿童园、动物园的相关文章
    

-   您可以将符合您预期的素材内容加入素材库
    

在搜索结果下方单击**\+ 加入素材库**按钮，弹出素材库选择浮层，勾选**公共素材库**或**我的素材库**，然后单击**添加**完成收藏。

-   随后您可以在【素材库】找到您刚刚搜索到并添加的素材内容
    

在**公共**页签的素材列表中，可看到已添加的素材"方案发布，青岛动物园拟搬迁！"等条目。
