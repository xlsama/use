### **PS：“无流式”和“有流式”区别：**

-   **无流式：**
    

之前的AI无流式生成路线为先基于文本生成，在文本生成结果上，进行多模态搜索，搜索到多模态内容后，再补充到生成结果当中，且AI搜索区域需要点击继续方可完成内容的生成。

-   **有流式：**
    

现如今的AI有流式生成路线分两种情况：

1.不需要澄清的情况下，AI思考和AI搜索自动跳过，无需停顿，直接生成结果

2.需要澄清情况下，仍然保持澄清后点击方可搜索和生成的逻辑

在有流式模式下，搜索界面顶部搜索栏右侧有蓝色 **Copilot** 开关（开启状态），搜索后自动依次完成 **Copilot 思考**（搜索需求分析）和 **Copilot 搜索**（搜索相关参考信息）两个阶段，无需停顿。搜索结果以卡片形式展示相关文章，下方紫色区域为 AI 生成的结构化摘要（包含若干要点），再下方依次展示**相关图片**和**相关视频**等多模态内容。
