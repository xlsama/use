# 没有思路，要谋篇布局

当你在撰写文章的过程中没有思路的时候，AI妙笔主要有三种方式为你提供思路。

**素材搜索**

-   如果您在撰写文章之前没有思路，那么可以试试先在“搜索素材”中搜索相关的内容；假设您需要写一篇有关“中国共同富裕”的文章。具体的操作如下图，在搜索框输入相关的内容，点击“搜索”。![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9922943071/p750189.png)
    

**说明**

搜索源为：夸克 ![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9922943071/p750185.png)

-   随后在弹出的搜索结果界面，您可以根据您的需求把相关合适的文章内容加入到素材库中。
    

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9922943071/p750188.png)

-   在弹出的界面中，勾选您需要添加到的素材库中即可。
    

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9922943071/p750187.png)

-   随后点击“素材库”，即可进入到对应的文章中。
    

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9922943071/p750186.png)

-   随后，返回至文章创作界面，点击“自定义素材”进行添加。
    

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9922943071/p750190.png)

### 分步骤撰写

当领导布置了一篇相关的短文任务，如果你对写作内容没有明确的思路，可以采用分步骤撰写的方式。

文章主题（prompt）

谈一谈中国的共同富裕

文章标题

中国的共同富裕理论

具体操作如下：

-   按系统要求填写【写作内容或主题】，输入【文章标题】、【写作场景】、【关键词】后点【下一步】
    

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9922943071/p750193.png)

-   查看生成的大纲
    

1).全部换一换

如果对生成的大纲不满意，只需点击“重新生成”即可生成新的大纲。

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9922943071/p750194.png)

2).编辑大纲

如果对生成的大纲相对满意，但是需要修改细节，那么只需点击编辑图标进行大纲细节的编辑。

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9922943071/p750191.png)

点击引用按钮图标可以添加外部内容或文章链接进行引用。

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9922943071/p750192.png)

-   查看生成的摘编
    

如果对生成的摘编不满意，只需点击“重新生成”即可进行优化操作。

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9922943071/p750196.png)

-   最后，点击“保存”即可保存此文章至【文档管理】中。
    

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9922943071/p750195.png)

## **选题热点**

在您按照需求用分步骤撰写完成文章后，如果您对内容还不是很满意，可以试一试“**选题热点”**功能。

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9922943071/p750199.png)

-   首先，点击“选题热点”，随后会弹出下图界面，在此界面下您可以看是否有符合文章主题的热点内容，点击下方加号即可添加。
    

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9922943071/p750198.png)

-   随后切换至【素材库】中，即可看到您刚添加到素材内容。
    

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9922943071/p750197.png)
