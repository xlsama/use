# 通义法睿

## **通义法睿产品介绍**

通义法睿是以千问为基座经法律行业数据和知识专门训练的法律行业大模型产品，综合运用了模型精调、强化学习、 RAG检索增强、法律Agent及司法专属小模型技术，具有回答法律问题、推理法律适用、推荐裁判类案、辅助案情分析、生成法律文书、检索法律知识、审查合同条款等功能。

通义法睿

法律咨询

提供法律问题的解答和建议的服务，包括对特定法律问题的理解、解释法律条文、提供可能的解决方案、建议采取的法律行动以及对相关法律程序的指导，增强训练的案由包括婚姻家事、民间借贷等案由。

案例检索

根据输入的内容在裁判文书库中进行类案检索，能够对检索结果自动分析，针对目标案例的争议焦点和核心事实给出相似案例的相关性分析。

法律法规检索

根据输入的内容在法律法规库中进行法规检索，实现了对法条语意理解基础上的智能检索，支持精准检索和模糊检索。

文本阅读

根据给定的文本进行法律层面的阅读理解，能够梳理法律实体与关系、法律事件时间脉络等。

文书生成

根据模板+要素的理解，结合用户的法律诉求自动生成法律文书，生成文书的输入既可以是案情描述，也可以是前置文书。

合同审查

根据审查规则审查涉及合同条款、法律合规性、业务逻辑及潜在风险等方面内容，支持自定义审查规则。
