# 系统配置

## **功能概述**

在系统配置中，用户可以根据需求和偏好进行个性化设置。系统配置包括干预规则和写作信源。

## **功能介绍**

### 干预规则

用户可以通过设置干预规则来调整AI妙笔的行为。干预规则可以用于限制或指导AI在生成内容时的行为，例如关键词干预、意图干预、网页干预。

新增干预规则时，需填写**规则名称**，选择干预类型后，为每个勾选的干预范围配置**回复方式**（如定向回复）及对应的回复内容，并设置**生效时间**（永久生效或指定时间范围）。

（1）关键词：支持填入关键词进行干预，一旦触发关键词，即可快速干预。关键词支持的干预范围包括写作提示词、写作结果、搜索词、搜索结果四大类。

在编辑关键词干预规则时，可为每个干预范围设置定向回复：**写作提示词**和**搜索词**回复"请更换输入内容试一试"，**写作结果**回复"涉及敏感词"，**搜索结果**回复"抱歉没有找到搜索内容相关的结果"。支持设置**生效时间**（如永久生效），并可在**干预关键词**区域添加多个关键词（如"政治"等）。

（2）意图：支持按照一句话、多句相似的话术进行干预，触发相似话术后，可快速干预。意图话术干预支持写作提示词、搜索词。

新增意图干预规则时，需配置**规则名称**、选择回复方式为**定向回复**并填写回复内容、设置**生效时间**（永久生效或指定时间范围），以及在**干预意图**中输入干预话术。

（3）网页链接：支持填入文章URL，在搜索召回和内容召回过滤对应文章的URL，不再列为参考文章，不参与搜索生成过程。干预范围支持参考文章和搜索来源。

新增干预规则时，还需配置**规则名称**（必填）和**生效时间**（可选**永久生效**或**指定时间范围**）。

### 写作信源

写作信源是指用于参考和借鉴的资源。用户可以选择不同的写作信源，如夸克搜索、央视网、新华网、人民网等，以便AI妙笔在生成内容时能够更好地理解和应用相关领域的知识。

默认配置中，**夸克通用搜索**的最大采信数量为 `20`，单击**编辑**按钮可修改权重配置。

**说明**

_注意：信源渠道网站是根据您提供的检索API而定。_

而权重配置会影响AI妙笔写作和搜索结果，权重（采信数量）越高，AI结果会参考的更多。

在**写作信源**页签的**最大采信数量**列中设置各数据源的采信数量，各数据源采信总和不超过50，不需要的数据源填0。设置完成后单击**保存**。
