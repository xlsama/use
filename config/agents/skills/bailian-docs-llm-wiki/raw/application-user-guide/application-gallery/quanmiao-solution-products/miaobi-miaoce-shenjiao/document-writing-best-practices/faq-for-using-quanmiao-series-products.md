## 什么时候用分步骤撰写

-   已经确定大纲框架时
    

当您对于需要创作的文章已经有大致的大纲安排时候，您可以优先使用分步骤撰写。

例如，撰写“中国共同富裕的理论”的文章，编辑人员已经明确有大致的方向，如从经济层面、社会层面、领导层面进行阐述。那么在这种时候是适合用分步骤撰写的，生成的结果也会无限接近编辑人员的预期效果。在分步骤撰写界面中，编辑人员可依次完成主题输入、大纲编辑等步骤，界面底部提供步骤导航按钮以控制创作流程。

-   当你想要更加可控生成文章时
    

分步骤撰写对比**直接生成**撰写的一个最大的特点就是**可控生成**，编辑人员可以对某个部分指定生成，精准确定大纲内容。这种可控生成的特性使得编辑人员可以更加精确地确定大纲内容。

分步生成模式包含 **主题**→**大纲**→**摘编**→**文章** 四个步骤，当前处于摘编阶段，系统根据大纲自动生成多条摘编素材供编辑人员展开查看，底部提供 **重新生成**、**上一步** 和 **下一步** 按钮，右侧为富文本编辑器区域，待生成文章后可直接编辑。

**说明**

此外，分步骤撰写还可以根据需求进行灵活调整和修改。如果您对某个部分生成的内容不满意或需要进一步修改，你可以随时进行编辑和调整，以达到你想要的效果。

## 什么时候自己引用素材

当编辑人员已经有相关的外部素材来源，包括不限于相关外部链接网址、纯文本内容等，在这样的条件下，优先自行添加引用素材内容。这么做的目的也是为了让生成的文章内容更可控，更符合编辑人员的预期。

单击**自定义素材**按钮，打开自定义素材对话框。对话框提供**素材库**和**搜索素材**两种查找方式，素材库下分为**公共**和**我的**两个分类。在素材列表中单击**+添加**将所需素材添加至右侧已添加列表（最多可添加 20 条），完成后单击**确认**。

**说明**

当编辑人员添加了相关指定引用素材，系统会有一定机率参考其数据；其在后续创作文章的权重也会比较大
