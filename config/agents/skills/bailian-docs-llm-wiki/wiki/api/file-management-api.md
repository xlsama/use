# file management api

百炼平台提供文件管理 API，用于管理上传至平台的文件资源。通过该 API，开发者可以完成文件的上传、查询、列举和删除等操作，为模型微调、知识库构建等场景提供数据准备支持。详细接口说明参见[文件管理](../../raw/model-api-reference/file-management-api.md)。

## 核心功能

文件管理 API 支持以下四类操作：

- **上传文件**：将本地文件上传至百炼平台，供后续模型训练或知识库使用。
- **查询文件**：根据文件 ID 获取单个文件的详细信息，包括文件名、大小、状态等元数据。
- **列举文件**：批量获取已上传文件的列表，支持分页查询。
- **删除文件**：移除不再需要的文件，释放存储空间。

如需了解各接口的请求参数与响应格式，请参考[文件管理](../../raw/model-api-reference/file-management-api.md)原始文档。

## 使用方式

开发者可通过 HTTP RESTful 接口调用文件管理功能。典型流程为：

1. 调用上传接口将训练数据或知识库文档上传至平台。
2. 通过列举或查询接口确认文件状态。
3. 在模型微调或知识库构建任务中引用已上传的文件。
4. 任务完成后，按需删除不再使用的文件。

## 注意事项

- 上传的文件需符合平台对格式和大小的限制要求。
- 删除文件前请确认该文件未被其他任务引用，以免影响正在运行的任务。
- 关于各接口的具体参数、错误码及调用限制，请以[文件管理](../../raw/model-api-reference/file-management-api.md)中的最新说明为准。
