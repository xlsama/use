# 电商文案智能可控生成

如果您需要撰写营销文案，通过使用阿里云百炼轻应用电商文案智能可控生成。基于全妙可控式生成大模型，深度融合用户输入与内置多元营销场景模板，智能输出商品标题、摘要及高转化营销文案。支持精准字数控制，确保内容精炼、卖点突出、风格统一，高效赋能营销内容创作，助力转化率全面提升。

**重要**

电商零售推广文案写作计费说明：

-   除全妙可控式LLM标准模型、全妙可控式LLM高级模型外，其他模型参与百炼新用户限时免费福利，更多关于新用户限时免费福利信息，请参见[新用户限时免费福利](https://help.aliyun.com/zh/document_detail/2793370.html)。
    
-   全妙计费文档详情可参见：[电商文案智能可控生成计费](https://help.aliyun.com/zh/model-studio/e-commerce-copy-intelligent-controllable-generation-billing)。

### **效果调试**

#### **基础营销文案写作**

> 创作类型：关于某产品的小红书笔记、商品使用测评文案、适合发朋友圈的推广营销文案、营销活动的宣传文案、某公司/产品的广告词、旅游类营销推广文章、其他类型的营销文案。

在**效果调试**页签，您可以参考以下步骤设置配置项，完成后单击**创作**。

-   **基础配置**：
    
    -   **创作类型**：选择需要创作的文体类型。支持选择的创作类型为7大类营销写作类型，分别是**关于某产品的小红书笔记**、**商品使用测评文案**、**适合发朋友圈的推广营销文案**、**营销活动的宣传文案**、**某公司/产品的广告词**、**旅游类营销推广文案**、**其他类型的营销文案**。
        
    -   **创作主题及参考素材**：您可以提供参考素材，指导AI的创作方向，创作出的内容会更符合您的预期。每个营销文体，都内置了输入要求，您只需按照输入要求填写相关信息，即可生成符合要求的营销文案。
        
    -   **选择模型**：支持选择**千问-Max**或**千问-Plus**模型。
        
    
-   **高级配置**：如果您对输出内容有定制化的要求，可通过高级配置进行控制，整体会与内置提示词合并后输入给大模型（如有冲突，以您的输入为准），生成符合您需求的内容。
    
    -   **补充其他要求**：
        
        -   **核心技能**：您可以输入对输出文案的某个部分的具体要求，比如：以一个问句的形式作为标题。
            
        -   **优先原则和限制约束**：您可以输入对整体输出文案的优先原则和限制约束内容，比如：所有的文案都不能出现错别字。
            
        
    -   **正面样本**：通过提供正面样本，让大模型更了解您的预期效果。即给到模型您想要的输入/输出正面样本，使模型可以按照您所要求的格式和风格进行输出。
