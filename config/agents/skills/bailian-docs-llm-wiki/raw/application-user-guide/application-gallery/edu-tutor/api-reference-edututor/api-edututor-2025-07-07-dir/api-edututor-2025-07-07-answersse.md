# AnswerSSE - 解题辅导

流式答题解析接口，客户输入需要解答题题目文本或图片，算法会返回题目详细的解答内容。

## 接口说明

开通 EduTutor 服务。本接口在公测阶段，每个账号每天可以免费调用 1000 次。
