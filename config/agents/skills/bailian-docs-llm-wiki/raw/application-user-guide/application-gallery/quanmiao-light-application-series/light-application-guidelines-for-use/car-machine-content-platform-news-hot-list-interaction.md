# 车机网络热点信息互动问答

针对汽车用户在驾驶过程中的收听新闻的需求，通过大模型实现新闻内容的自动加工、分类聚合，富媒体呈现、交互式播报、个性化呈现，打造智能座舱新闻聚合播报产品。该产品可用于对内容平台的海量内容进行聚类加工，提炼出核心话题及聚合摘要，成为读者对内容平台进行速览、导读的指导。

**重要**

车机/内容平台新闻热榜互动应用限时免费，额度用完后再按实际调用模型对应的输入、输出Token以后付费方式来计费。

-   更多关于新用户限时免费福利信息，请参见[新用户限时免费福利](https://help.aliyun.com/zh/document_detail/2793370.html)。
    
-   关于Token的计算方法和模型的计费详情，请参见[计费项与定价](https://help.aliyun.com/zh/model-studio/miaosou-miaodu-api-billing)。

### **效果调试**

#### **第一步：配置播报单内容**

1.  查看官方聚合新闻播报单：**车机网络热点信息互动问答**应用基于主流互联网平台数据，系统每4小时更新一次，单击**默认播报单**进行查看。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2112853471/p935632.png)**聚合新闻播报单**从科技、娱乐、经济、社会、体育、教育、旅游、文化8个频道分析全部热点新闻，每条播报单的信息包含热点名称、新闻数量、官方热度指数。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2112853471/p935633.png)
    
2.  自定义聚合新闻播报单：
    
    配置每个频道生成的新闻数量，可以选择全量生成（默认值：**全部**），也可以自定义选择热度前20以内的新闻。
    

#### **第二步：聚合摘要的写作风格**

1.  单击![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1439580371/p868138.png)图标，打开**聚合摘要的写作风格**开关。
    
    如果官方聚合的播报单在输出格式、字数、风格人设等方面不符合要求，可自定义聚合播报单。
    
2.  填写**聚合摘要的写作风格和要求**的提示词prompt
    
    -   如果不进行prompt和热点话题配图修改，则按照默认聚合新闻播报单的形式输出；
        
    -   如果在人设风格、输出格式、字数、配图等方面不符合要求，可自定义聚合播报单的输出形式。
        
        -   支持更改prompt指令，调整聚合新闻播报单的输出形式，包含任务描述、人设风格、输出格式、字数等
            
    
    **示例**prompt
    
    #【任务描述】
    
    你是一个资深新闻编辑，给你一个热点事件和与之相关的多个文章标题/摘要等。请你从这些多篇文章内容中，总结出关于该热点事件最主要、最重要的几个核心讨论主题，每个主题总结一个名称（名称需是一个完整通顺的句子），每个话题做一个简介，字数在100字左右，（介绍该主题下的关键内容）。
    
    #【热点事件】
    
    {hotTopic}
    
    #【文章主题标题摘要列表】
    
    {titleSummaries}
    
    #【人设风格】
    
    请以汽车车机智能车机助理的风格。
    
    #【输出格式】
    
    不要输出额外内容。不要返回任何其他东西。请不需要虚构任何数据。
    
    参考【人设风格】，严格按照如下格式输出：
    
    核心主题名称：xxx
    
    核心主题概要：xxx
    
    核心主题名称：xxx
    
    核心主题概要：xxx

如需要测试个性化推荐效果，请加钉钉群（群号：116015001424） ，我们会以离线形式支持您进行测试。

#### **第四步：互动问答**

> **针对新闻内容进行问答**开关默认打开，您可向播报单提问，针对新闻播报单的细节进行问答。

1.  设置**互动的数据范围**，支持多选。
    
    -   **仅针对聚合新闻播报单内已包含的信息进行回答**：用户输入query后，接口可以输出聚合新闻的片段，比如某个频道下的所有热点、某个频道的某个热点；或者围绕新闻播报单主动输出一个query，勾起用户对话互动的兴趣，用户回应后会播报聚合新闻片段。
        
    -   **允许针对新闻的原始素材和互联网开放域进行搜索后回答**：用户输入query后，输出热点对应的原始新闻里的细节，并给出3个推荐query。
        
2.  在**指令（Prompt）**文本输入框中输入指令，例如任务描述、上下文信息、人设风格等。您也可以使用系统预置的参考示例。
    
3.  给答案配图，您可以选择**不配图**，也可以自定义选择配置1～3张图。
    
4.  单击**确认设置检查结果**，针对新闻内容进行问答。
    
    选择**AI先抛出话题**或者**等待用户先提问，AI再回答**。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2112853471/p935644.png)
    

### API示例

单击**确认设置检查结果**，可生成对应的API，您可复制API代码进行调用。

调用官方聚合新闻播报单的用户，API会调用已经通过离线方式准备好的数据。数据每4小时更新一次，时间大致在00:30、04:30、08:30、12:30、16:30、20:30。请根据更新时间，按需调用。
