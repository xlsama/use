# AI妙策

## **功能概述**

在文章创作过程中，用户可以借助AI妙策来辅助创作。AI妙策主要集中在聚合热点话题榜和平台话题榜两个方面。

![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5148919071/p771423.png)

## **具体功能**

### **话题分析**

AI妙策中的话题分析可以作为创作的灵感来源，通过网页视角选题、热门视角选题、时效视角选题、新颖视角选题，可以抓住当前受关注的热点问题。在创作过程中，可以根据AI妙策分析话题的视角分布，挖掘有潜力的文章角度，从而使文章更具有吸引力和时效性。

![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/6148919071/p771424.png)

-   网页视角选题
    

把某个选题下对应的所有新闻下的所有网友观点聚合起来，并统计里面主要的观点/态度。把占前80%的态度逐个列出，支持往下生成选题策划。

![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/6148919071/p771426.png)

-   热门视角选题
    

热门视角选题是当前话题下热度最高的多条新闻切入选题，可以帮助作者洞察何种选题视角有最高的热度。

![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5148919071/p771429.png)

-   时效视角选题
    

时效视角选题是当前话题下最近前10条发表的新闻切入选题，可以辅助作者掌握最近的新闻风向。

![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/5148919071/p771430.png)

-   新颖视角选题
    

是指目前在该选题下，尚无媒体用过的切入点视角，比较新颖的视角。

![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/6148919071/p771431.png)

### **选题策划**

选题策划是指基于用户选择的视角选题方向、新闻内容素材，生成该选题的策划提案，快速满足用户提案、框架构思的需要。支持提案内容包含主题、主题摘要、段落大纲、大纲要求描述，满足用户提案的基础需求。支持策划提案以思维导图、Word形态进行下载等管理操作；还可联动AI妙笔分步骤撰写，利用大模型进行辅助创作。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/6148919071/p771433.png)
